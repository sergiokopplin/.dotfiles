Ducky One 2F Media Configs

RESET
- Left Windows Key + Arrow Down Key

Set Profile to 2
- Fn + Alt + 2

CAPS LOCK to fn
- Fn + Alt + K for 5 seconds
- Caps Lock once until BLUE LED
- L_CTRL until RED LED
- ESC

Set Play/Pause to Fn + Space
- Fn + Alt + Tab for 3 seconds
- Fn + Space -> target
- Fn + Windows Key + D -> source
- Fn + Alt
- Fn + Alt + Tab for 1 second

Set Next Track to Fn + Arrow Right
- Fn + Alt + Tab for 3 seconds
- Fn + Arrow Right -> target
- Fn + Windows Key + F -> source
- Fn + Alt
- Fn + Alt + Tab for 1 second

Set Previus Track to Fn + Arrow Left
- Fn + Alt + Tab for 3 seconds
- Fn + Arrow Left -> target
- Fn + Windows Key + G -> source
- Fn + Alt
- Fn + Alt + Tab for 1 second

Set Volume Up to Fn + Arrow Up
- Fn + Alt + Tab for 3 seconds
- Fn + Arrow Up -> target
- Fn + Windows Key + B -> source
- Fn + Alt
- Fn + Alt + Tab for 1 second

Set Volume Down to Fn + Arrow Down
- Fn + Alt + Tab for 3 seconds
- Fn + Arrow Down -> target
- Fn + Windows Key + C -> source
- Fn + Alt
- Fn + Alt + Tab for 1 second
