Tested on Ubuntu 19.04

- **Git** Required;
- **SSH** configured. `ssh-keygen -t rsa -b 4096 -C "sergiokopplin@gmail.com"`;
- **Chrome** Installed;

```
git clone https://github.com/sergiokopplin/.dotfiles.git ~/.dotfiles &&
cd ~/.dotfiles &&
./install
```

Also, but manually:

- Dracula Chrome DevTools

Gnome Extensions:
- Remove Alt+Tab Delay
- Dash to Panel
- Dash to Dock
- Panel OSD

Keyboard:
- Key to choose the 3rd level: Any Win
- Screenshot of an area Ctrl + Shift + Alt + 3
- Show the overview Alt + Space
- Maximize Window Shift + Ctrl + Alt + F
- View Split on left Shift + Ctrl + Alt + {
- View Split on right Shift + Ctrl + Alt + }
